# 客户交易行为分析 Skill

## 技能描述

客户交易行为分析技能。当需要分析客户的基金/投资产品交易行为时使用此技能。处理包含多年、多资产类别的客户交易数据和市场指数数据的Excel文件，生成全面的行为诊断报告。使用中文进行所有分析和输出。

## 概述

本技能提供了一套使用交易数据诊断客户投资行为的系统性框架。

## 数据内容说明

所提供的Excel文件包括5页数据，分别对应5大类资产：

| Sheet名称 | 对应指数 | 指数涨跌幅字段 |
|----------|---------|--------------|
| 国内主动权益产品交易行为 | 上证综指 | 上证综指当日涨跌幅 |
| 国内权益指数产品交易行为 | 上证综指 | 上证综指当日涨跌幅 |
| 海外权益产品交易行为 | 纳斯达克 | 纳斯达克当日涨跌幅 |
| 固收产品交易行为 | 中证全债 | 中证全债当日涨跌幅 |
| 货币基金交易行为 | 中证全债 | 中证全债当日涨跌幅 |

每页数据字段结构：
- 日期
- 指数收盘点位
- 指数当日涨跌幅
- 申购金额、申购笔数
- 定投金额、定投笔数
- 赎回金额、赎回笔数

资产类别：国内主动权益、国内权益指数、海外权益、固收、货币基金

## 核心目标

基于提供的多年度、多资产类别的客户交易数据与市场指数，进行系统性、多维度的行为诊断。目标不仅是描述行为，更要识别行为模式背后的动因、评估其有效性、量化潜在问题，并生成高度个性化的优化建议。

## 分析步骤与指标计算

### 第1步：数据预处理与增强

标记市场环境：根据上证综指、纳斯达克、中证全债三个指数的月度形态，判定每个月处于"上涨市"、"下跌市"或"震荡市"，并合并连续同向月份形成主要趋势段。

判定标准（月度形态分析法）：

1. 按月度分组统计：将数据按年-月分组，计算每个月的涨跌幅、振幅、日收益率标准差（波动率）
2. 月度形态判定规则：
   - 上涨市：月度涨跌幅 > 3%
   - 下跌市：月度涨跌幅 < -3%
   - 震荡市：-3% ≤ 月度涨跌幅 ≤ 3%
3. 月度波动率计算：月度波动率 = 该月日收益率的标准差 × √22（年化），用于辅助判断市场活跃程度
4. 合并连续同向月份：当连续2个月以上表现相同形态时，合并为主要趋势段，记录合并后的时间段、累计涨跌幅、持续月数
5. 特殊形态标记：
   - 快速下跌：月度跌幅 > 8%
   - 强劲上涨：月度涨幅 > 8%
   - 高波动月：月度波动率 > 25%

注意：每次分析择时行为时，应使用对应资产类别所关联的指数进行市场环境判定。

重要：月度形态分析仅作为内部分析支撑，用于判断客户交易行为与市场环境的关系，不直接输出到最终报告中。

### 第2步：统计交易行为汇总指标

按以下5大类汇总统计客户的交易行为概况：

资产类别映射：
- 国内主动权益 → 国内主动权益产品交易行为
- 国内权益指数 → 国内权益指数产品交易行为
- 海外权益 → 海外权益产品交易行为
- 固收 → 固收产品交易行为
- 货币基金 → 货币基金交易行为

汇总指标（按资产类别）：
| 指标 | 计算方式 |
|------|---------|
| 申购总额 | SUM(申购金额) |
| 申购总笔数 | SUM(申购笔数) |
| 定投总额 | SUM(定投金额) |
| 定投总笔数 | SUM(定投笔数) |
| 赎回总额 | SUM(赎回金额) |
| 赎回总笔数 | SUM(赎回笔数) |
| 净申购总额 | 申购总额 + 定投总额 - 赎回总额 |

输出格式：以表格形式展示各大类产品的汇总指标，并简要总结所提供时间段内各大类产品的总体交易情况，包括：
- 哪些类别净申购最高（资金流入最多）
- 哪些类别净赎回最高（资金流出最多）
- 哪些类别交易最活跃（笔数最多）
- 各大类产品的配置比重（按净申购金额计算）

### 第3步：分类行为特征量化分析

根据不同资产类别的特性，采用差异化的分析重点。

#### A. 权益类产品交易行为分析

（国内主动权益、国内权益指数、海外权益）

A1. 定投与单笔申购配合分析：
| 指标 | 计算方法 | 判定逻辑 |
|------|---------|---------|
| 定投占比 | 定投金额 / (申购金额+定投金额) | 判断定投是否是主要投资方式 |
| 定投+单笔配合度 | 有定投的月份中，同时有单笔申购的月份数 / 总月份数 | 是否用单笔配合定投做长期投资 |
| 低位加仓识别 | 单笔申购时前置20日涨幅 < μ-0.5σ（市场回调中加仓） | 是否有逆向投资意识 |
| 高位止盈识别 | 赎回时前置20日涨幅 > μ+1.5σ（市场高位赎回） | 是否懂得止盈 |
| 定投标的集中度 | 各类权益产品定投金额占比 | 是否专注某一类产品 |

洞察：判断客户是否将定投作为减少市场波动干扰的工具，同时用单笔申购在低位加仓、高位止盈。

A2. 择时行为分析：
| 指标 | 计算方法 | 判定逻辑 |
|------|---------|---------|
| 追涨识别 | 申购时前置20日涨幅 > μ+1.5σ | 市场上涨后追买 |
| 逆向布局识别 | 申购时前置20日涨幅 < μ-0.5σ | 市场回调时逆势买入 |
| 杀跌识别 | 赎回时前置20日涨幅 < μ-1.5σ | 市场下跌后恐慌赎回 |
| 趋势赎回识别 | 赎回时前置20日涨幅 > μ+0.5σ | 上涨后赎回（可能止盈或止损） |

A3. 定投纪律性分析：
| 指标 | 计算方法 | 判定逻辑 |
|------|---------|---------|
| 定投执行率 | 实际定投月数 / 应定投月数 | 是否坚持长期定投 |
| 中断次数 | 定投从有到无的切换次数 | 定投是否有中断 |
| 市场波动期中断率 | 指数单日涨跌幅>3%时暂停定投的次数 | 是否在市场恐慌时中断定投 |
| 低位增投识别 | 市场低位时定投金额是否增加 | 定期不定额意识 |

#### B. 固收/货币类产品交易行为分析

B1. 强制储蓄属性分析：
| 指标 | 计算方法 | 判定逻辑 |
|------|---------|---------|
| 定投储蓄率 | 固收/货币定投金额 / 总定投金额 | 定投是否作为储蓄手段 |
| 定投稳定性 | 固收/货币定投金额的月度波动率（CV值） | 是否定期小额投入（类似储蓄） |
| 最低余额占比 | 期间固收/货币最低净申购金额占比 | 是否保持最低储蓄底线 |
| 配置周期 | 固收/货币连续净申购/净赎回的最长持续月数 | 是长期配置还是短期行为 |

洞察：判断固收/货币类定投是否作为强制储蓄手段，保持长期稳定的小额投入。

B2. 股债轮动分析：
| 指标 | 计算方法 | 判定逻辑 |
|------|---------|---------|
| 高位转换识别 | 上证综指>μ+1.5σ时，赎回权益+申购固收的次数/金额 | 风险规避意识 |
| 低位转换识别 | 上证综指<μ-1.5σ时，赎回固收+申购权益的次数/金额 | 逆向布局意识 |
| 无转换识别 | 股市大幅波动时，固收/货币无明显净申购/净赎回 | 是否忽视资产再平衡 |

洞察：判断客户是否在股市风险偏高时将权益转换为固收/货币，在股市低位时逆向布局。

#### C. 无定投客户的行为特征与建议

C1. 客户类型识别：
| 类型 | 特征 | 分析重点 |
|------|------|---------|
| 纯单笔投资型 | 无定投，仅有大额单笔申购/赎回 | 分析其择时能力，判断是否需要建议定投 |
| 观望型 | 交易频率低，资金主要在货币基金 | 分析是否有机会成本，建议资产配置 |
| 频繁交易型 | 无定投但主动交易频繁 | 分析交易成本损耗，建议定投纪律 |

C2. 针对性建议（无定投客户）：
- 纯单笔投资型："您目前主要采用一次性单笔投资方式，建议可考虑将部分资金改为定投，以平均成本、分散风险，特别是在市场波动期间，定投可以帮助您避免择时错误。"
- 观望型："您目前大量资金闲置在货币基金中，建议根据个人风险偏好，将部分资金通过定投方式配置到权益或固收产品中，提高资金效率。"
- 频繁交易型："您主动交易较为频繁，可能产生较高的交易费用，且容易受市场情绪影响。建议设定定投计划，用纪律化的投资方式替代主观择时。"

### 第4步：综合分析、客户画像与建议生成

#### A. 生成多维客户画像标签

综合以上所有维度，基于客户实际交易行为数据，自定义生成画像标签。画像标签应由数据驱动，不应预先限定选项，应根据客户的真实行为特征精准描述。

画像维度与描述方法：

交易风格：根据客户的交易频率、定投占比、主动交易比例等数据，用一句话精准描述其交易习惯。

配置特征：根据各类资产的净申购金额占比和交易集中度，用一句话描述其配置偏好。

择时特征：根据前置窗口分析和择时胜率数据，用一句话描述其择时能力与倾向。

行为稳定性：根据定投中断次数、季度间交易波动等数据，用一句话描述其行为稳定性。

#### B. 深度行为总结

优势与亮点：基于数据明确指出做得好的地方（如：在市场连续下跌期间坚持定投、在股债之间进行再平衡操作等）

核心问题与风险：
- 行为偏差：是否存在过度自信、损失厌恶、处置效应等
- 成本损耗：高频交易导致的估算费用损耗
- 收益损耗：因不当择时（如"高买低卖"）导致的潜在收益损失
- 配置风险：资产过度集中或错配带来的风险收益比不佳

#### C. 生成个性化、可执行的优化建议

建议必须与上述分析结论直接挂钩，并尽可能量化：

针对交易习惯：国内权益、固收、海外频繁申购、赎回，高位大额申购国内权益、权益指数、海外权益，低位暂停定投等行为均不可取，客户如有此类行为则需要指出

针对资金流向：资金是否频繁在各类资产中流动，若是则可建议客户根据个人风险偏好做好资产配置，避免频繁交易

针对择时行为："数据显示，您过去N次权益申购中，有M次属于'追涨'，且其中P次申购后市场出现回调。建议：1）将单次大额择时申购改为定期不定额（下跌多买）；2）设置严格的交易纪律（如指数市盈率低于历史N%分位时增加投资）。"

## 最终输出格式

以面向客户的口吻进行表达，称谓用"您"。

报告结构：
```
交易行为深度分析报告

分析周期：XXXX年XX月 - XXXX年XX月

一、核心数据概览
[表格：各资产类别汇总指标，包括申购总额/笔数、定投总额/笔数、赎回总额/笔数、净申购总额]

二、客户行为特征全景画像
- 交易风格：（根据实际数据自定义描述）
- 配置特征：（根据实际数据自定义描述）
- 择时特征：（根据实际数据自定义描述）
- 行为稳定性：（根据实际数据自定义描述）

三、权益类产品交易行为分析
3.1 定投与单笔申购配合分析
3.2 择时行为分析
3.3 定投纪律性分析（适用于有定投客户）

四、固收/货币类产品交易行为分析
4.1 强制储蓄属性分析
4.2 货币基金分析

五、关键行为分析发现
5.1 值得保持的优势
5.2 亟待改进的风险点

六、量化行为有效性评估
- 择时胜率：XX%

备注：择时胜率计算规则
- 择时胜率 = 择时成功的交易笔数 / 总择时交易笔数 × 100%
- 择时成功定义：交易方向与后续市场走势一致
  - 申购后20日内市场上涨 → 成功（买入时机正确）
  - 赎回后20日内市场下跌 → 成功（卖出时机正确）
  - 反之则为择时失败

七、具体优化建议与行动计划
7.1 针对有定投客户的建议
7.2 针对资产配置的建议
```

## 使用方法

当用户请求客户交易行为分析时：
1. 请求或定位包含交易数据的Excel文件
2. 加载数据并识别工作表结构
3. 按照上述框架执行分析
4. 按照指定格式生成综合报告
5. 用数据驱动的洞察和可执行的建议呈现发现
